CURD Operations using AngularJs and Hibernate

Implemented CURD operations using AngularJS(1.6v). Performed Single Page Routing(SPA) using angular-route.js(1.6v) which is compatable with angularjs(1.6) Captured the form data from registration form by JSON string with help of $http(post) method. Intracted with RestFul Webservices at the backend used hibernate for save, update, delete, and insert the record in database.
